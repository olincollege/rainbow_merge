Rainbow Merge

The purpose of this project was to use Python object-oriented programming to create a merging game.

Check out our Github repository and README file.

Game Controls

A Key: Move piece to the left.
D Key: Move piece to the right.
S Key: Drop the block.

Scoring

Get The Highest Score!

Merge the same color blocks to increase your score!
But be careful! Only blocks that merged horizontally will count!

Red = 2 points
Orange = 4 points
Light Orange = 8 points
Yellow = 16 points
Green = 32 points
Green Blue = 64 points
Grey Blue = 128 points
Blue = 256 points
Purple = 512 points
Black = 1024 points
